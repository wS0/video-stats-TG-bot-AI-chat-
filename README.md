# video-stats-TG-bot-AI-chat-

## Запуск
0. Requirements для запуска обоих файлов: pip install sqlalchemy psycopg2-binary aiogram openai
1. Кладём данные в БД.
Перед запуском скрипта на преобразование исходного json в содержимое таблиц БД следует в файле json_to_psql.py сконфигурировать подключение к БД: имя пользователя БД, пароль пользователя БД, хост СУБД, порт СУБД.
2. Запускаем телеграм-бота.
Для запуска бота запускаем файл bot.py, в который предварительно записываем параметры подключения к БД точно так, как в первом пункте, api-ключ для использования нейросети (в этом случае chatgpt, поэтому и api-ключ от open ai), токен для использования телеграм-бота.
3. Переходим в телеграм и сразу задаем боту вопросы, на команду /start не реагирует.

## Описание решения
В каждом запросе дополнительно к пользовательскому вопросу объясняем нейросети схему БД и ожидания, проиллюстрированные примерами. Полученный SQL применяем на БД и если в ответе получаем число, то отдаем его пользователю, если не получаем, то просим переформулировать вопрос.
